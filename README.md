- Node (v12.0.0+)

### âš™ï¸ How To Use

  

1. Clone this repository

  

```bash

git clone https://github.com/yasinatesim/next-starter-kit.git

```


2. Install the project dependencies

```bash

yarn install

```

**For Development**

```bash

yarn dev

```

  

**For Production Build & Build Start**

```bash

yarn build

```

  

and

  

```bash

yarn start

```

  

**For Export [SSG](https://www.staticgen.com/)**

```bash

yarn export

```

  

**For Lint & Format**

```bash

yarn lint

yarn format

```
